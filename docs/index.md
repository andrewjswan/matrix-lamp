## Description
A compilation of firmware adaptations based on the original idea of Gyver Lamp, for ESPHome.

## Firmware
You can install the firmware using [ESPHome](https://esphome.io/) using one of the ready-made configuration files or use [**MatrixLamp - ESP Web Tools**](firmware.html) - a convenient tool for installing and updating the firmware of ESP32 devices in the browser.
